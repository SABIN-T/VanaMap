# PlantFinder Project

This project is organized into two main applications:

## 1. Frontend (`/frontend`)
- **Tech**: React, Vite, TypeScript.
- **Run**: 
  ```bash
  cd frontend
  npm run dev
  ```
- **Deploy**: Upload this folder to Vercel/Netlify.

## 2. Backend (`/backend`)
- **Tech**: Node.js, Express, MongoDB.
- **Run**:
  ```bash
  cd backend
  npm start
  ```
- **Deploy**: Upload this folder to Render/Heroku.
